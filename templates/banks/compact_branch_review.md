# Compact Branch Review Template

Use when the report style is dense, branch-review oriented, and expects concise paragraphs.

## Style

- Keep each subsection to one or two paragraphs.
- Prefer “总体看 / 需关注 / 后续需核验” phrasing.
- Use tables for metric evidence; avoid long prose restating every row.

## Required Caveat

```text
上述分析仅基于已提供或已抽取财务数据，不替代最终授信审批结论。
```
